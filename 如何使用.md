# 如何使用

本目录是发布到 GitHub 的安全沙箱，只包含可公开的模板文件。

## 文件说明

| 文件 | 说明 |
|---|---|
| `README.md` | 项目介绍文档 |
| `preferences-template.md` | 偏好配置模板（不含个人信息） |

## 首次使用步骤

1. 将 `preferences-template.md` 复制为 `preferences.md`
2. 按你的实际情况填写：目标职位、地点、薪资、偏好等
3. 配置猎聘 MCP
4. 开始搜岗

## ⚠️ 重要提醒

**永远不要把以下文件上传到 GitHub：**

- ❌ `preferences.md` - 包含你的薪资期望、地点偏好等个人信息
- ❌ 任何 `job*.md` 文件 - 包含你投递过的岗位记录
- ❌ `优化简历/` 目录 - 包含你的个人简历和照片
- ❌ `job_apply_log.md` - 包含你的投递历史
- ❌ `.env` - 包含 API Keys

这些文件应该只存在于你本地的工作目录中。

## 安全建议

- 根目录 `.gitignore` 已配置好大部分隐私文件
- 如要推送到自己的仓库，建议 **单独给 publish 目录建仓库**
- 或者只手动上传这两个文件到 GitHub，不要整目录推送
